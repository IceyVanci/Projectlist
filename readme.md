### OneFrame

**仓库**: [IceyVanci/OneFrame](https://github.com/IceyVanci/OneFrame)

一款简洁优雅的图片边框添加工具，为您的照片自动添加精美的底部边框，并智能显示相机 EXIF 信息。

### ColorPhoto

**仓库**: [IceyVanci/colorphoto](https://github.com/IceyVanci/colorphoto)

图片颜色提取工具

这是一个基于 Electron 的图片颜色提取工具，灵感来自在十年前曾经流行过的某种图片贴纸。从导入的图片中分析出图片的主要颜色，然后以色块在图片上显示。拖动色块调整大小，不管是边框，注释，还是NSFW（？），创建属于你的专属设计！

### ColorPhoto-s

**仓库**: [IceyVanci/colorphoto-s](https://github.com/IceyVanci/colorphoto-s)

ColorPhoto 的 Tauri 版本

基于 IceyVanci/colorphoto 移植的 Tauri 版本

### forza-paint-calculator-cuda

**仓库**: [IceyVanci/forza-paint-calculator-cuda](https://github.com/IceyVanci/forza-paint-calculator-cuda)

将图片几何化为旋转椭圆并使用CUDA加速的工具。为了加速forza-painter的计算过程而设计，但并未成功，目前软件计算效率远低于forza-painter。感谢Xiaomi MiMo-V2-Pro在公测的第一周提供了免费的API token。原始代码的GPU重新实现，无法解决能量算法在GPU上的显存瓶颈。test分支使用Minimax M2.7进行了分析优化，仍然未解决计算速度问题
